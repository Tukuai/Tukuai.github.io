正则表达式

regular Expression

为什么需要正则表达式

复杂文本的处理

正则表达式的优势和用途

强大的文本处理工具，基本上的编程语言，数据库，文本编辑器都支持

正则表达式的定义

正如名字一样描述了一个规则，通过这个规则可以匹配一类字符串
学习正则表达式很大程度上就是学习正则表达式的语法规则

##开发中使用正则表达式
1.分析所要匹配的数据，写出测试用的典型数据
2.在工具软件中进行匹配测试
3.在程序中调用通过测试的正则表达式
