HTML

概念：最基础的网页开发语言

Hyper Text Markup Language 超文本标记语言
超文本：
超文本是用超链接的方法，将各种不同空间的文字信息组织在一起的网状文本。
标记语言：
由标签构成的语言。<标签名称>如HTML，XML
标记语言不是编程语言

快速入门：

  1. HTML文档后缀名：.html或者.htm
  2. 标签分为
    • 围堵标签：有开始标签和结束标签，如< html> < /html>
    • 自闭和标签：开始标签和结束标签在一起。如< br/>
  3. 标签可以嵌套
    需要正确嵌套,不能你中有我，我中有你
    错误：< a>< b>< /a>< /b>
    正确< a>< b>< /b>< /a>
  4. 在开始标签中可以定义属性，属性是由键值对构成，值需要用引号(单双都可)引起来
  5. html不区分大小写，但建议使用小写

标签学习：

1.文件标签：构成html的最基本的标签

html：html文档的根标签
head：头标签。用于指定html文档的一些属性，引入外部的资源
title:标题标签
body：体标签
< !DOCTYPE>?:定义文档类型

2.文本标签：和文本有关的标签

注释：<! –注释内容–>
< h1> to < h6>：标题标签
h1到h6字体大小逐渐递减
< p>：段落标签
< br>：换行标签
< hr>：展示一条水平线:属性:color,width：宽度,size：高度,align：对齐方式center,left,right
< b>：字体加粗
< i>：字体斜体
< center>:字体居中
< font>：字体标签:属性:,color:颜色,size：大小,face：字体

属性定义：
color：
1.英文单词
2.rgb：0~ 255
3.#值1值2值3:00~ FF之间
width：
1.数值：width=’20’,数值的单位，默认是px
2.数值%：占比相对于父元素的比例

3.图片标签：和图片有关的标签

< img src=””/>
src:相对路径, ./：代表当前目录(默认), ../：上一级目录
alt:规定图像的代替文本
< img src=”timg.jpg” wdith=”400” height=”300”/>

4.列表标签：

  1. 有序列表
    ol：order list
    li
  2. 无序列表
    ul:unorder list
    li

5.链接标签：

< a>:定义一个超链接
属性：
href：指定访问资源的URl（统一资源定位符）
target：打开方式,_ self表示在本页面内打开，_ blank表示打开新的页面
功能：

  1. 可以被点击：样式
  2. 点击后跳转到href指定的url
    若只想保留第一个功能：< a href=”javascript:void(0);” id=”del”>删除子节点< /a>

6.块标签(和CSS结合使用，本身没有什么作用，但可用于CSS控制样式)

span：文本信息在一行内展示，行内标签，内联标签，不会换行
div：每一个div占满一整行，块级标签，会换行。

7.语义化标签:html5中为了提高程序的可读性，提供的标签

< header>
< footer>

8.表格标签：

< table>：定义一个表格
< tr>:行标签
< td>:单元格 属性:colspan:列的长度，rowspan：行的宽度，都是用来合并
< th>:表头单元格
< caption>:定义表格标题
< thead>:表示表格的头部分，增强可读性
< tbody>：体
< tfoot>：脚

9.表单标签

概念：用于采集用户输入的数据的。用于和服务器进行交互。

标签：form用于定义表单的。可以定义一个范围，范围代表采集用户数据范围。

  • 属性：action：指定提交的url，method：指定提交方式
  • method分类：一共七种，常用的有两种
    • get：
      1.请求参数会在地址栏中显示。
      2.请求参的长度是有限制的
      3.不太安全
    • post：
      1.请求参数不会在地址栏中显示，会封装在请求体中。
      2.请求参数的长度是没有限制的
      3.较为安全
      表单项的数据要想被提交，必须指定其name属性

表单项标签：

  1. input：可以通过type属性值改变展示的样式
    • type属性：

      1.text：文本输入框，默认的

      placeholder：提示输入信息，如请输入用户名

      2.password：密码输入框
      3.radio：单选框

      要想让多个单选框实现单选的效果，则name属性必须一样。
一般会给每一个单选框提供value属性，指定其被选后提交的值
checked属性可以指定默认值

      4.checkbox：复选框

      要想让多个复选框实现复选的效果，则name属性必须一样。
一般会给每一个复选框提供value属性，指定其被选后提交的值
checked属性可以指定默认值

      5.file：文件选择框
      6.hidden：隐藏域，用于提交一些信息
      7.按钮

      submit：提交按钮。可以提交表单
button：普通按钮
image：图片提交按钮,src：图片路径

      8.color：取色器
      9.date：日期

    • label：指定输入项的文字信息
      label的for属性一般会和input的id属性对应，如果对应了，则点击label会让input输入框获取焦点
  2. select：下拉列表
    子元素：option，列表项
  3. textarea：文本域
    cols：指定列数，每行有多少字符
    rows：指定行数，会自动扩容
