JS-BOM

概念

Browser Object Model 浏览器对象模型
将浏览器的各个组件部分封装成对象

组成

Window：窗口对象

  1. 创建
    Window对象不需要创建，可以直接window.方法名();
    也可以直接使用方法名；
  2. 方法
    • 与弹出框有关的方法
      alert()：显示带有一段消息和一个确认按钮的警告框。
      confirm()：显示带有一段消息以及确认按钮和取消按钮的对话框。如果用户点击确定按钮,则方法返回true,取消则返回false；
      prompt()：显示可提示用户输入的对话框。返回值获取用户输入的值。
    • 与打开与关闭有关的方法
      close():关闭当前浏览器窗口,谁调用就关谁
      open():打开一个新的浏览器窗口或查找一个已命名的窗口。返回新打开的窗口的对象。
    • 与定时器有关的方法
      setTimeout()：在指定的毫秒数后调用函数或计算表达式。;参数：1.JS的代码片段，或者方法对象2.毫秒值; 返回值：返回setTimeout对象的编号。用于取消定时器。
      clearTimeout()：取消由 setTimeout()方法设置的 timeout。参数：参数setTimeout对象的唯一编号。
      setInterval()：按照指定的周期（以毫秒计）来调用函数或计算表达式。参数：1.JS的代码片段，或者方法对象。2.毫秒值
      clearInterval() 取消由 setInterval() 设置的 timeout。
  3. 属性
    • 获取其他BOM对象：
      history
      location
      Navigator
      Screen
    • 获取DOM对象
      docunment=window.document
  4. 特点
    Window对象不需要创建，可以直接window.方法名();也可以直接使用方法名；

History：当前页面历史记录对象

1.获取
1.window.history
2.history
2.方法
back()：加载 history 列表中的前一个 URL。
forward()：加载 history 列表中的下一个 URL。
go()：加载 history 列表中的某个具体页面。
3.属性
length:返回当前窗口的历史列表中URL个数

Location：地址栏对象

1.创建(获取)
window.location或者直接location
2.方法
reload()：重新加载当前文档。刷新
3.属性
href：设置或返回完整的 URL。

Navigator：浏览器对象
Screen：显示器屏幕对象
