    JavaScript

    概念:一门客户端脚本语言，运行在客户端浏览器中的。每一个浏览器都有JavaScript的解析引擎，脚本语言：不需要编译，直接就可以被浏览器解析执行。
    功能:增强用户和HTML的交互过程，可以来控制HTML，让页面有一些动态效果，增强用户的体验。
    JavaScript发展史：

    1. 1992年，Nombase公司开发出第一门客户端脚本语言，专门用于表单校验，命名为：C–，后来更名为ScriptEase。
    2. 1995年，Netscape（网景）公司，开发了一门客户端脚本语言：LiveScript。并且95年Java1.0面世。后来网景请来Sun公司的专家，修改LiveScript，命名JavaScript，想蹭Java的热度。
    3. 1996，微软推出IE浏览器，抄袭了JavaScript开发出了JScript语言。
    4. 1997，ECMA（欧洲计算机制造协会）统一所有的客户端脚本语言，根据JavaScript制定了ECMAScript。所以JavaScript=ECMAScript+JavaScript自己的（BOM+DOM)

    ECMAScript：客户端脚本语言的标准

    基本语法

    1. 与HTML的结合方式
      • 内部JS
        定义< script>< /script>标签，标签体就是js代码
      • 外部JS
        定义< script>标签，通过src属性引入外部的js文件

    注意：可以定义多个，可以定义在任意位置，不过会影响执行顺序。
    2. 注释
    单行注释：//
    多行注释/ * * /
    3. 数据类型
    * 原始数据类型：（基本数据类型）
    number:数字。 整数，小数，NAN（not a number）
    string：字符串’abc’ “abc”
    boolean：true或false
    null：一个对象为空的占位符
    undefinrd：未定义。如果一个变量没有初始化值，则会被默认为undefined。
    * 引用数据类型：对象
    4. 变量
    一小块存储数据的内存空间，java语言是强类型的语言，而JavaScript是弱类型的语言
    强类型：在开辟变量存储空间时，定义了空间将来存储的数据类型。只能存储固定的数据类型
    弱类型：在开辟空间时，不定义将来的存储类型，可以存放任意类型的数据
    语法：var 变量名=初始化值；typeof可以查出是哪种数据类型。
    5. 运算符
    * 一元运算符：只有一个运算数的运算符
    ++，–，+（正号）
    在JS中，如果运算数不是运算符所要求的类型，那么JS会自动将运算数进行类型转换
    其他类型转number
    string转number：按照字面值转换。如果字符串字面值不是数字，则转为NaN；
    boolean转number：true转为1，false转为0；
    * 算术运算符：
    +，-，* ， /,/ %…
    * 比较运算符：
    < > >= <= == ===(全等于)
    比较方式：
    1. 类型相同：直接比较
    字符串：按照字典顺序比较，按位逐一比较，直到得出大小。
    2. 类型不同：先进行类型转换再比较
    ===：再比较之前，先判断类型，如果类型不一样，则返回false。
    * 逻辑运算符
    &&：与（短路） ，||：或（短路），！：非
    其他类型转boolean
    1.number：0或NaN为假，其他为真
    2.string：除了空字符串，其他都是true
    3.null&undefined：都是false
    4.所有对象都为true
    * 三元运算符
    ？ ：
    例：a>b?1:0
    6. 流程控制语句
    > 1.if else
    2.switch，在java中switch可以接受的数据类型：byte short int char 枚举 字符串，在js中，什么类型都可以接收
    3.while
    4.do while
    5.for
    7. 特殊语法
    * 在JS中如果一行只有一条语句，则分号可以省略，不推荐
    * 变量的定义可以使用var关键字，也可以不使用，也不推荐；用：定义的变量为局部变量，不用：定义的变量为全局变量

    基本的对象

    Function：函数（方法）对象

    1. 创建
      • var fun=new Function（形式参数列表，方法体） 不建议使用
      • function 方法名称（形式参数列表）
        {
        方法体
        }
      • var 方法名=function（形式参数列表）
        {
        方法体
        }
    2. 方法
    3. 属性
      length：形式参数的个数
    4. 特点
      • 方法定义时形参的类型不用写
      • 方法是一个对象，如果定义一个名称相同的方法，则会覆盖
      • 在js中方法的调用只与方法的名称有关，和方法的参数列表无关
      • 在方法声明中有一个隐藏的对象（数组）arguments，封装所有实际参数
    5. 调用

    Array：数组对象

    1. 创建：
      • var arr=new array（元素列表）；
      • var arr=new Array（默认长度）
      • var arr=[元素列表]；
    2. 方法
      join：将数组中的元素按照指定的分隔符拼接为字符串
      push：向数组的末尾添加一个或更多元素，并返回新的长度。
    3. 属性
      length
    4. 特点
      • JS中，数组的元素类型可变，如var arr=[1，”abc”,true];
      • JS中数组的长度是可变的。

    Data：日期对象

    1. 创建：var date=new Date（）；
    2. 方法：toLocaleString()：根据本地时间格式，把 Date 对象转换为字符串。
      getTime()：返回 1970 年 1 月 1 日至今的毫秒数。

    Math：数学对象

    1. 创建：Math对象不用创建直接使用。Math.方法名();
    2. 方法：
      random()：返回 0 ~ 1 之间的随机数。包含0，不包含1。
      ceil(x)：对数进行上舍入
      floor(x)：对数进行下舍入。
      round(x)：把数四舍五入为最接近的整数。
    3. 属性：PI

    RegExp：正则表达式对象：定义字符串的组成规则
    正则表达式

    1. 单个字符:[]
      如：[a]：a， [ab]：a或者b ，[a-z]a到z
      \d：单个数字字符[0-9]
      \w:单个单词字符[a-zA-Z0-9_]
    2. 量词符号
      ？：表示出现0次或1次
      星号：表示0次或多次
      +：出现一次或多次
      {m,n}:数量大于等于m小于等于n；m要是缺省，最多n次；n要是缺省，最少m次。
    3. 开始符号：^
    4. 结束符号:$

    正则对象

    1. 创建：
      • var reg=new RegExp(“正则表达式”)；
      • var reg=/正则表达式/
    2. 方法
      test（参数）：验证指定的字符串是否符合正则定义的规范
      var flag=reg.test("  ...  ");返回true或者false；

    Global

    1. 特点：全局对象，这个Global中封装的方法不需要对象就可以直接调用，方法名（）；
    2. 方法：
      encodeURI() 把字符串编码为 URL。
      decodeURI() 解码某个编码的 URL。
      encodeURIComponent() 把字符串编码为 URL组件。和上面的区别：编码的字符更多，冒号斜杠之列的也会编码。
      decodeURIComponent() 解码一个编码的 URL 组件。
      parseInt():将字符串转为数字。逐一判断每一个字符是否是数字，直到不是数字为止，将前面的数字部分转换为数字。
      isNaN():判断一个值是否是NaN，NaN六亲不认，连自己都不认，NaN参与的==比较，全部为false
      eval()：计算 JavaScript 字符串，并把它作为脚本代码来执行。
      var jscode="alert(123)";
eval(jscode);
    3. urL编码
      把字符的每个字节前用%
      如王佳豪=%E7%8E%8B %E4%BD%B3 %E8%B1%AA
      这里使用的utf-8编码，所以3个汉字有9个字节
    4. 例：
      var str=”王佳豪”;
      var encode=encodeURI(str);
      var s=decodeURI(encode);

      • 获取元素对象
      • 使用innerHTML属性修改标签体内容

    事件简单学习

    功能：某些组件被执行了某些操作后，触发某些代码的执行。
    如何绑定事件：

    1. 直接在HTML标签上，指定事件的属性（操作），属性值就是js代码。(耦合度比较高，不推荐使用)
      onclick
      < img id=”qq” src=”q.jpg” onclick=”alert(“….”)”>
      < img id=”qq” src=”q.jpg” onclick=”fun()”>

    2. 通过JS获取元素对象，指定事件属性，设置一个函数。
      < img id=”qq” src=”q.jpg” >
      var qas=document.getElementById(“qq”);
      qas.onclick=f;
